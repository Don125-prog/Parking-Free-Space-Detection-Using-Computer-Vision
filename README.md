# Определение свободных и занятых парковочных пространств

Этот репозиторий содержит учебный проект для дисциплины "Компьютерное зрение". 
В данной работе исследуется применимость You Only Look Once (YOLO) для решения задач автоматического определения и классификации состояния парковочных мест. Обучение модели проводилось на собранном и размеченном наборе данных, включающий в себя реальные видеозаписи парковочных зон, синтетически сгенерированные изображения, а также отобранные из открытых источников изображения парковочных мест. Для решения задачи использовались модели YOLOv11, YOLOv12 обученные в течении 100 эпох и проведен сравнительный анализ эффективности моделей

# Набор данных

Для обучения и тестирования рассматриваемых в данной работе нейросетей использовались наборы данных, как локальные, собранный автором, так и данные из открытых источников. После завершение процедуры разметки, было аннотировано 11 420 парковочных мест, из которых было 6 930 аннотировано занятыми и 4 490 свободными. Общий размер размеченного набора данных составил 2 071 изображений, и был выложен на HuggingFace в разных форматах (coco, yolo):
https://huggingface.co/datasets/Don125/Parking-Free-Space/tree/main
